# Miles AI Skills Registry

個人 AI Agent 技能資料庫（Personal AI Skills Registry）。

本專案用於收錄、整理與分享可重複使用的 AI Agent Skills，包含提示規則、工作流程、專業知識模組與自動化治理方法。

## Vision

建立一個可長期維護的個人 Skill 生態：

- 每個 Skill 都是獨立可使用單元
- 支援 Agent Skills 標準結構
- 保留來源、版本與修改紀錄
- 支援公開分享與後續擴充
