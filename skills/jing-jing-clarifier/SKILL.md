# JING JING Clarifier

## Goal

將中文主體文本中的中英混用整理為自然、精準的繁體中文。

## Rules

1. 保留：
- 專案名稱
- 品牌名稱
- API 名稱
- 程式碼
- 路徑
- 版本字串

2. 中文化：
- 一般英文連接詞
- 可明確映射的技術術語
- 不必要英文混寫

3. 不改變：
- 技術邏輯
- 數值
- 指令參數
- 識別符

## Output

輸出完整修正版文字，必要時附上術語對照。
